# MBTI ไม่แม่น แต่แซวแม่น 🔮

แบบทดสอบบุคลิก MBTI สายฮาภาษาไทย เล่นจบใน 24 ข้อ เลือก "ภาพที่เป็นคุณที่สุด" แล้วรับการ์ดผล 1 ใน 16 ไทป์ พร้อมกราฟเรดาร์ ค่าพลัง คู่ที่เข้ากัน และปุ่มแชร์

**เล่นเลย:** https://nuttapon.github.io/mbti/

## ฟีเจอร์

- **24 คำถาม 4 ตัวเลือก** แบ่ง 6 ข้อต่อแกน (E/I, S/N, T/F, J/P) ทุกตัวเลือกเป็นภาพวาด SVG ทำมือ
- **การ์ดผลแบบจัดเต็ม** — ชื่อไทป์, hook กวน ๆ, กราฟเรดาร์ 4 แกน, ค่าพลังเฉพาะไทป์, สกิลเด่น/พลังหมดเมื่อ, และ 3 คู่ที่อยู่ด้วยแล้วเวิร์ก
- **ป้ายความหายาก** — บอกว่าไทป์นี้พบราว ๆ กี่ % ของคนทั้งโลก (อิงสัดส่วนประชากร MBTI โดยประมาณ)
- **บันทึก progress อัตโนมัติ** — ปิดแท็บกลางคันแล้วกลับมาเล่นต่อจากข้อเดิมได้ (`localStorage`)
- **ลิงก์ผลแบบแชร์ตรง** — `?type=INFP` เปิดหน้าผลของไทป์นั้นทันที ไม่ต้องเล่นซ้ำ
- **แชร์ + บันทึกเป็นรูป** — มือถือแชร์การ์ดเป็นรูปผ่าน native share sheet, เดสก์ท็อปแชร์ลิงก์ (unfurl ด้วย OG image) หรือคัดลอกลิงก์
- **ลูกเล่น** — อนิเมชันสลับคำถามแบบสไลด์, คอนเฟตตีตอนเฉลยผล
- **เล่นด้วยคีย์บอร์ดได้** — กด `1`–`4` เลือกตัวเลือก, ลูกศรเลื่อนโฟกัส; ตัวเลือกเป็น `radiogroup` รองรับ screen reader
- **เคารพ `prefers-reduced-motion`** — ปิดอนิเมชันและคอนเฟตตีให้อัตโนมัติ

## รันในเครื่อง

เป็นเว็บ static ล้วน (vanilla JS + ES modules) ไม่มี build step แต่ ES modules ต้องเสิร์ฟผ่าน http (เปิดไฟล์ตรง ๆ ด้วย `file://` จะติด CORS):

```bash
python3 -m http.server 8000
# เปิด http://localhost:8000
```

## เทสต์

```bash
npm test   # node --test (ตรรกะ scoring, ข้อมูล quiz, โครง HTML, share payload)
```

## โครงโปรเจกต์

```
index.html               โครงหน้า 3 จอ (intro / quiz / result) + OG meta tags
style.css                ธีม neo-brutalist สีพาสเทล
js/
  app.js                 ตัวคุมหน้าจอ, persistence, deep link, แชร์, คอนเฟตตี
  quiz-data.js           คำถาม, โปรไฟล์ 16 ไทป์, ค่าพลัง, ความหายาก, คู่ที่เข้ากัน
  quiz-engine.js         รวมคะแนน → ไทป์ + สถิติเรดาร์ (pure functions)
  quiz-session.js        เก็บคำตอบรายข้อ (Map)
  choice-illustrations.js  ภาพ SVG ทำมือของทุกตัวเลือก
  confetti.js            คอนเฟตตี canvas แบบไม่มี dependency
tests/                   node:test
assets/
  og-image.png           รูป preview ตอนแชร์ลิงก์ (1200×630)
  og-template.html        เทมเพลตไว้ render og-image.png ใหม่
```

### สร้าง OG image ใหม่

`assets/og-image.png` สร้างจาก `assets/og-template.html` ด้วยการเปิดในเบราว์เซอร์ที่ขนาด 1200×630 แล้ว screenshot (Chrome DevTools / headless ก็ได้) เซฟทับไฟล์เดิม

## หมายเหตุเรื่องความแม่น

ป้าย % ความหายากและผลทั้งหมดเป็นแนวสนุก ๆ ไม่ใช่เครื่องมือวินิจฉัยทางจิตวิทยา — เล่นขำ ๆ ได้เลย 😄
